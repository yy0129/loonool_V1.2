基础页 · FAQ
常见问题 · FAQ
1. 什么是图片审核空间（Image Review Space）？
图片审核空间是一个专门用来“在图片对外使用前做决策”的协作区域。
在同一个 Space 中，团队可以围绕多个任务（Task）上传候选图片、进行讨论、使用 AI 工具生成审核依据，并最终选出可以放心使用的图片。
2. 什么是任务 Task？
Task 表示一次具体的“视觉决策场景”，例如：
  • 本次首页 Banner 主图选图
  • 某款新品包装要上架前的图片审核
每个 Task 下可以放 1〜6 张候选图片、一个讨论流（Discussion）和一组审核依据（Evidence）。
3. 为什么一个任务最多只能放 6 张图片？
LOONOOL 希望每次决策保持“可对比、可讨论、可拍板”，而不是无限堆图。
6 张是一个兼顾：
  • 屏幕上同时对比的可读性
  • 团队讨论时能集中注意力
  • 导出与回顾时的清晰度
的上限。
当第 7 张图片加入时，系统会自动创建一个新的 Task，保持每个 Task 的决策粒度清晰。
4. 什么是 Evidence / Add Evidence / Analysis Tools？
Evidence（审核依据） 是记录“为什么这样选图”的证据链，包含：
  • AI 工具分析结果（风险 / 风格 / 市场）
  • 团队上传的规则文档、截图、外部链接等
  • 系统生成的关键决策摘要

Add Evidence 是上传正式证据的入口，例如：
  • 平台图片规范 PDF
  • 竞品截图
  • 外部链接（自动抓取标题 + 摘要）

Analysis Tools 是一组 AI 分析工具，分为：
  • 风险工具（Risk Tools）
  • 风格工具（Style Tools）
  • 市场工具（Market Tools）
用户在当前 Task 的候选图基础上运行工具，得到结构化结果，并一键保存为 Evidence。
📘 使用建议
建议的使用顺序是：
  1. 在首页创建一个 Space（审核空间）；
  2. 将本轮要决策的候选图上传到 Task 中；
  3. 邀请团队成员加入空间，一起评论与讨论；
  4. 使用 Analysis Tools 生成关键审核依据，并通过 Add Evidence 补充外部证据；
  5. 选择 Final 图片，并导出结果或分享在线预览链接。
